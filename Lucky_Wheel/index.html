抽獎公平性說明

本網站的中獎結果並非由轉盤動畫決定，而是在您按下「開始抽獎」時，便先完成隨機演算。系統使用 Web Crypto API 的 crypto.getRandomValues 產生密碼學等級亂數，並以拒絕取樣（rejection sampling）均勻抽取索引，降低取模偏差，使每一位參加者在每一輪被抽中的機率相同。若設定中獎者移出名單，則採不放回抽樣；同一輪抽出多人時，亦避免重複中獎。轉盤旋轉僅供視覺呈現，不會改變最終結果。
